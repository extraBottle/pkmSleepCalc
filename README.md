- 좋은 캠프 티켓 사용 여부 선택 기능 추가
- 힐러 포켓몬 설정 (라이트/프로) 기능 추가
- 수면 시간 선택 기능 추가

- 사이드바로 다른 웹페이지 연결 기능 추가
